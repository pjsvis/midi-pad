# VR Effect Plugins

## Introduction

- Set up 4X instruments with 2X joystick controls
- IE 16 macros in Banks 3 and 4

- Bank3 => CC 46 - 53
- Bank4 => CC 54 - 61

- J1A A: 46, E: 47
- J1B W: 48, D: 49

- J2A A: 50, E: 51
- J2B W: 52, D: 53

- J3A A: 54, E: 55
- J3B W: 56, D: 57

- J4A A: 58, E: 59
- J4B W: 60, D: 61
